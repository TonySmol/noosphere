# 🌌 NOOmium — P2P Thought Resonator

**NOOmium** превращает ваши мысли в смысловые векторы и ищет им созвучие в вашей
личной базе и среди открытых мыслей других людей.  
Работает прямо в браузере, не требует сервера, хранит данные локально и общается
через Nostr-протокол.

---

## ✨ Возможности

- **Смысловой поиск (AI)**  
  Локальная модель машинного обучения (Transformers.js) превращает каждую мысль в
  эмбеддинг и находит близкие идеи по косинусному сходству.  
  Если модель не загрузилась — автоматически включается лексический режим.

- **Три дорожки резонанса**  
  - **Моё** — сильные совпадения в вашей базе.  
  - **Мир** — открытые мысли других людей (P2P через Nostr).  
  - **Озарения** — близкие, но неожиданные связи, вдохновение.

- **Приватность и обмен**  
  Мысли можно сохранять лично или делать открытыми для сети.  
  Чужие мысли нельзя присвоить — их можно прочитать, понять и написать своё.

- **Offline-first**  
  База данных на IndexedDB с автоматическим переходом в in‑memory‑режим, если
  IndexedDB недоступен.  
  Лексический поиск работает вообще без сети.

- **Интернационализация**  
  Русский и английский языки, переключение без перезагрузки.

- **Тёмная и светлая темы**  
  Уважительный контраст, минималистичный дизайн.

- **Экспорт / импорт базы**  
  Вся ваша коллекция мыслей упаковывается в JSON‑файл, который можно сохранить
  или передать.

- **Умная P2P‑сеть**  
  Защита от спама релеев: rate‑limit с экспоненциальной отсрочкой, ограничение
  количества ответов на запрос, дедубликация событий.

- **Мобильный‑фёрст**  
  Адаптивная вёрстка, поддержка safe‑area, жесты свайпа для быстрой отправки.

## 🚀 Быстрый старт

1. Скачайте файл [`noomium.html`](./noomium.html).
2. Откройте его в любом современном браузере (Chrome, Firefox, Safari, Edge).
3. При первом запуске потребуется интернет для:
   - загрузки модели ИИ (~20 МБ, кешируется браузером);
   - подключения к Nostr‑релеям.
4. После загрузки модели всё работает полностью локально, включая поиск по смыслу.

**Размещение на сервере:**  
Просто положите HTML‑файл в корень вашего сайта. Никакого бэкенда не нужно.

---

## 🧠 Как это работает

1. Вы вводите мысль.
2. Модель `all-MiniLM-L6-v2` (запущенная локально) превращает текст в
   384-мерный вектор.
3. Локальная база данных (IndexedDB) ищет заметки с максимальным косинусным
   сходством.
4. Одновременно вектор отправляется в Nostr-сеть (событие kind 21000) — другие
   участники в той же комнате видят запрос и могут вернуть свои открытые мысли
   (событие kind 21001).
5. Результаты раскладываются по трём дорожкам в зависимости от степени совпадения.

Весь код работает на стороне клиента — ваши мысли не покидают ваше устройство,
если вы не делаете их открытыми.
